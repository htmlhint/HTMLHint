---

import { Badge } from '@astrojs/starlight/components';

Inline style cannot be used.

Level: <Badge text="Warning" variant="caution" />

## Config value

- `true`: enable rule
- `false`: disable rule

### The following pattern is considered a rule violation:

```html
<div style="color:red"></div>
```
